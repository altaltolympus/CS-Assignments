Meeting The Brief

BR1: Embedded System Design

The embedded system I made takes digital input from a button which activates and deactivates it, as well as analogue input from an environmental sensor.

As well as sending the data from the sensor to a connected computer and displaying it as status bars on the screen, the system can sound an alarm if the conditions detected by the sensor are too extreme.

BR2: Data collection

The system uses an environmental sensor to measure both soil humidity and water level. It then sends this data to a computer over USB connection, where it is stored in a .csv file.

BR3: Process simulation

I have implemented a function to test the operation of the system. This function, triggered by button press, simulates water level and soil humidity readings gradually becoming very high before falling back down again.
